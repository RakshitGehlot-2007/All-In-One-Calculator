# All-In-One-Calculator
All-in-One Calculator is a modern web-based calculator built with HTML, CSS, and JavaScript. It combines multiple tools in a single application, including Basic, Scientific, Unit Converter, Discount, Percentage, Loan, Tip, BMI, and Date calculators, include calculation history, memory functions, dark/light mode, and a responsive user interface.
Features
Basic Calculator
Scientific Calculator
Unit Converter
Discount Calculator
Percentage Calculator
Loan Calculator
Tip Calculator
BMI Calculator
Date Calculator
Calculation History
Memory Functions (MC, MR, M+, M-)
Dark/Light Theme Toggle
Responsive Design
